# qt-cli

> 一个专注于 react + typescript + electron 开发的工作流

# Install

```
yarn add global qingtian-cli
```

or

```
npm install qingtian-cli -g
```

# Contains

-   web 以及 electron 项目初始化 ✅
-   常用代码生成 ✅
-   一键进入 webpack 开发模式和打包模式 ✅
-   内置 typescript 支持 ✅
-   内置 sass 支持 ✅
-   内置 postcss 支持 ✅
-   webpack 打包分析 ✅
-   css modules ✅
-   支持多页应用构建 ✅

# Commands

-   **new**： 创建 web 或 electron 项目
-   **dev**： 启动项目开发
-   **build**： 生产环境构建
-   **generate**： 常用代码生成
